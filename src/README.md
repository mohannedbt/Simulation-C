# `src/` — Source files

This directory contains the project's primary C sources and example demos. Key files:

- `Engine.c` — Solar-system demo using raylib (visualization + physics).
- `particles.c` — N-body / 3-4 body demo (raylib + raygui UI).
- `Particles_V2.c` — alternative particle-system implementation.
- `Osci/` — small oscillator / spring demo sources (see `src/Osci/README.md`).
- `Rubix/` — 3D rotating cube example (see `src/Rubix/README.md`).

How to build: see repository root `README.md` for build commands. Prefer compiling individual source files into `bin/` executables.
